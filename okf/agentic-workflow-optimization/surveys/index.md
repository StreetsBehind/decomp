# Surveys & critique

The landscape papers that map the field, plus the one skeptical paper that bounds it.

# Landscape

* [From Static Templates to Dynamic Runtime Graphs (IBM)](ibm-workflow-optimization-survey.md) - Frames workflows as agentic computation graphs; organizes the field by when/what/which-signal. The best single map (2026, recent — verify).
* [Self-Evolving Agents — What/When/How/Where](self-evolving-agents-survey-what-when-how-where.md) - TMLR-2026 four-interrogative taxonomy; the cleanest model for placing this program (recent — verify).
* [A Comprehensive Survey of Self-Evolving AI Agents (EvoAgentX)](evoagentx-self-evolving-survey.md) - Organized by optimization scope; its prompt-optimization sub-taxonomy situates GEPA.

# Critique

* [Inefficiencies of Meta Agents for Agent Design](inefficiencies-of-meta-agents.md) - The counterweight: meta-agent search is inefficient, low-diversity, and rarely cost-justified (break-even only past 15,000 deployment examples). Read before committing M5 budget.
